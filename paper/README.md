# Law Delegation Paper Draft

This folder is an Overleaf-ready two-column LaTeX paper package.

## Main files

- `main.tex`: paper source
- `references.bib`: bibliography
- `figures/hitl_workflow.tex`: original vector human-in-the-loop diagram
- `figures/workflow_dag.tex`: original vector workflow-architecture diagram
- `figures/Workflow.png`: product screenshot of the workflow graph
- `figures/AI_Benchmark.png`: product screenshot of benchmark-comparison cards
- `figures/README.md`: figure notes

## Compile

On Overleaf, set `main.tex` as the main document and compile with a standard LaTeX engine.

Locally with Tectonic:

```bash
cd paper
tectonic main.tex
```
